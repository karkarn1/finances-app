# Finances App

Personal finance management web application built with React 18+, TypeScript, Material-UI, and Redux Toolkit. Track your net worth, manage accounts, monitor investments, plan expenses and income, and achieve financial goals with AI-powered insights.

### Prerequisites
- **Node.js** >= 18.0.0
- **Yarn** >= 1.22.0

### Installation

1. **Clone and navigate to the project:**
```bash
cd finances-app
```

2. **Install dependencies:**
```bash
yarn install
```

3. **Start the development server:**
```bash
yarn dev
```

The application will open at [http://localhost:5173](http://localhost:5173)
